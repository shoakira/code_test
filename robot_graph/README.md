# 📊 robot_graph

Contains scripts and data for graph-based representations of robot behaviors and dependencies.

## 🚀 Scripts

- `PDDL_graph.py`: PDDL-related graph generation.
- `rdf2neo4j.py`: Script to transform RDF data for Neo4j consumption.

## 📈 Visualizations & Data

- `dependency_visualization.svg`: SVG visualization of component dependencies.
- `cypher_queries.txt`: Useful Cypher queries for querying the robot graph.
- `fig2.md`, `test.md`: Documentation and Markdown-based graph descriptions.
