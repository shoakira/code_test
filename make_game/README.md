# 🎮 make_game

Contains scripts and resources related to game development.

## 🚀 Scripts

- `galaga.py`: A Galaga-style shooting game.
- `galaxian.py`: A Galaxian-style shooting game.
- `invader.py`: Classic Space Invaders clone.
- `othello.py`: Othello (Reversi) implementation.
- `ping_pong.py`: Simple Ping Pong game.
- `quiz.py`: A quiz game script.
- `super_mario.py`: A Super Mario-style platformer demo.

## 🖼️ Assets

- `images/`: Directory containing game assets.
- Various PNG files for bullets, invaders, and spaceships.
