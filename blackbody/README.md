# 🌡️ blackbody

Physics-based simulations and visualizations related to blackbody radiation and thermodynamics.

## 🚀 Physics Visualizations

- `fig_planch_function.py`: Plots showing the Planck function.
- `fig_heat_capacity.py`: Visualizing heat capacity curves.
- `spectrum.py`, `spectrum_3comp.py`: Spectrum analysis tools.
- `vibration_mode.py`, `vibration_mode_2d.py`: Wave and mode simulations.

## 📈 Outputs

Contains numerous generated PNG files (e.g., `blackbody_radiation.png`, `heat_capacity_curve.png`) illustrating various physical laws and comparisons.
