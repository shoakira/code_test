# 🤖 openai_test

Contains scripts for testing and demonstrating various OpenAI API features.

## 🚀 Features

- `openai_json_format.py`: Demos for forcing JSON output from models.
- `openai_translator.py`: Translation features using GPT models.
- `paper_review.py`: Automated paper review and analysis script.
- `ner_test.py`: Named Entity Recognition (NER) testing.
- `openai_vllm_test.py`: Integration test with vLLM and OpenAI-compatible APIs.
- `openai_o1_demo.py`: Demo for the OpenAI o1 models.
