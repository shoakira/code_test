# 🧠 RL_code

Contains code related to reinforcement learning algorithms and environment testing.

## 🚀 Algorithms

- `RLPolicyGradient.py`: Implementation of a Policy Gradient algorithm.
- `RLgym.py`: Scripts for interacting with OpenAI Gym (or similar) environments.

## 💾 Saved Models & Media

- `policy_net_checkpoint.pth`: Saved PyTorch model weights for the policy network.
- `video/`: Directory containing demo videos of trained agents.
