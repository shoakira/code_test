# 🔍 gemini_embedding2_test

Experiments and tests using the **Gemini Embedding 2** model for multimodal similarity and retrieval.

## 🚀 Key Scripts

- `extract_with_embedding.py`: Calculates and visualizes cosine distances for all PDF pages relative to a query.
- `embedding_test.py`: General test script for multimodal embedding capabilities.
- `image_matrix_similarity.py`: Visualizes similarity heatmaps between multiple images.
- `mixed_matrix_similarity.py`: Cross-modal similarity analysis between images and text.
- `scene_similarity_test.py`: Fine-grained semantic comparison for complex scenes.

## 🛠️ Traditional CV Comparison

- `extract_pdf_pages.py`: Pixel-perfect search using Template Matching.
- `extract_pdf_pages_sift.py`: Geometric pattern matching using SIFT.

## 📊 Visualizations

- `pdf_distance_matrix.png`: Heatmap generated by `extract_with_embedding.py`.
- `mixed_similarity_matrix.png`, `similarity_matrix.png`, etc.
